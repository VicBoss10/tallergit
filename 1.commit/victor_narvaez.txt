Ejercicio 1

1.1. Preguntas

1. ¿Cómo se inicializa un repositorio local? (que comando se debe ejecutar?)
Respuesta: Se usa el comando "git init" esto cuando ya se esta dentro 
de la carpeta requerida

2. ¿Cómo hago para que un directorio deje de ser controlado por git? (que comando se debe ejecutar?)
Respuesta: Usamos el comando git rm -r --cached <nombre_del_directorio> esto hará que git deje de 
controlar el directorio mas los archivos no serán alterados.

3. Si agrego un archivo a un directorio que ya está siendo controlado por git, ¿está siendo controlado por git?
Respuesta: No, al agregar archivos o alterar el codigo se debe hacer un "add" con el nombre del archivo y luego se hace un 
commit para que quede registrado

4. ¿Qué comando se utiliza para agregar un archivo al repositorio local?
Respuesta: Se utiliza el comando "git add <nombre_del_archivo>"

5. ¿Cómo determino que archivos fueron modificados? (que comando se debe ejecutar?)
Respuesta: Para ver lo archivos modificados se usa el comando "git status"

6. ¿Qué comando se utiliza para hacer un commit? 
Respuesta: Se usa el comando "git commit -m "mensaje_del_commit" 

7. En sus propias palabras, ¿qué es un commit?
Respuesta: Se podria decir que es una confirmación de que se quiere guardar o conservar un cambio hecho
